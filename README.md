# 随迹冒险

这是一个面向 Android 的原生小应用：每天打开后先布置任务，应用会自动给出一个随机生活挑战；选择任务和预计时长后，就进入倒计时。

## 当前版本包含

- 今日任务输入
- 任务重要性标注：关键主线、重要支线、轻量探索
- 每天自动生成一个随机挑战
- 可更换随机挑战
- 选择任务后设置 5 到 120 分钟倒计时
- 暂停、继续、提前完成
- 当天任务保存在手机本地，第二天自动刷新随机挑战

## 在手机上安装

1. 在电脑安装 Android Studio。
2. 用 Android Studio 打开这个文件夹：`C:\Users\Jingmin Luo\Documents\SuiJi APP for Android`
3. 等待项目同步完成。如果提示安装 Android SDK 35 或 Gradle，按提示安装即可。
4. 手机打开“开发者选项”和“USB 调试”，用数据线连接电脑。
5. 在 Android Studio 顶部选择你的手机，点击运行按钮。

也可以在 Android Studio 中选择 `Build > Build Bundle(s) / APK(s) > Build APK(s)`，生成调试 APK 后复制到手机安装。第一次安装时，手机可能会要求允许“安装未知来源应用”。
